# TUF Interactive Calendar

A polished React calendar component inspired by a physical wall calendar design. It includes:

- responsive hero image card and month panel
- date range selection on the calendar grid
- integrated notes section with localStorage persistence
- mobile-friendly layout

## Run locally

1. Install dependencies

```
npm install
```

2. Start the dev server

```
npm run dev
```

3. Open the local URL shown in the terminal.
